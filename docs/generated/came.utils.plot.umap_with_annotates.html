came.utils.plot.umap_with_annotates¶

came.utils.plot.umap_with_annotates(adt, text_ids, color=None, anno_df=None, text_col=None, index_col=None, anno_fontsize=12, ax=None, fp=None, **plkwds)¶

Plot the UMAP embeddings and annotate the names of the given points

Parameters

  • adt (AnnData) –

  • text_ids (Sequence) – ids of the points to be annotated, corresponding to anno_df.index or anno_df[index_col] (if index_col is given)

  • color (Optional[str]) – a column name in adt.obs

  • anno_df (Optional[DataFrame]) – use adt.obs as the default annotations.

  • text_col (Optional[str]) – used as the annotation texts. if specifies, should be a column name in anno_df.

  • index_col (Optional[str]) – used for index the points, corresponding to text_ids if specifies, should be a column name in anno_df.

  • anno_fontsize (int) – the fontsize of the annotations

  • ax – the plot ax

  • fp – file path for saving the plot.

  • plkwds – other key-word-args for sc.pl.umap

Returns

Return type

ax
