came.pipeline.preprocess_aligned¶

came.pipeline.preprocess_aligned(adatas, key_class, df_varmap_1v1=None, use_scnets=True, n_pcs=30, nneigh_scnet=5, nneigh_clust=20, deg_cuts={}, ntop_deg=50, ntop_deg_nodes=50, key_clust='clust_lbs', node_source='hvg,deg', ext_feats=None, ext_nodes=None)¶

Packed function for process adatas with aligned features (i.e., one-to-one correspondence).

Processing Steps:

  • align variables

  • preprocessing

  • candidate genes (HVGs and DEGs)

  • pre-clustering query data

  • computing single-cell network

Parameters

  • adatas ([<class 'anndata._core.anndata.AnnData'>, <class 'anndata._core.anndata.AnnData'>]) – A pair of sc.AnnData objects, the reference and query raw data

  • key_class (str) – the key to the type-labels, should be a column name of adatas[0].obs

  • df_varmap_1v1 (Optional[DataFrame]) – dataframe containing only 1-to-1 correspondence between features in adatas; if not provided, map the variables of their original names.

  • use_scnets (bool) – whether to use the cell-cell-similarity edges (single-cell-network)

  • n_pcs (int) – the number of PCs for computing the single-cell-network

  • nneigh_scnet (int) – the number of nearest neighbors to account for the single-cell-network

  • nneigh_clust (int) – the number of nearest neighbors to account for pre-clustering

  • deg_cuts (dict) – dict with keys ‘cut_padj’, ‘cut_pts’, and ‘cut_logfc’, used for filtering DEGs.

  • ntop_deg (Optional[int]) – the number of top DEGs to take as the node-features

  • ntop_deg_nodes (Optional[int]) – the number of top DEGs to take as the graph nodes

  • key_clust (str) – where to add the per-clustering labels to the query data, i.e., adatas[1].obs. By default, it’s set as came.pipeline.KEY_CLUSTER

  • node_source (str) – source of the node genes, using both DEGs and HVGs by default

  • ext_feats (Optional[Sequence]) – extra variables (genes) to be added to the auto-selected ones as the observation(cell)-node features.

  • ext_nodes (Optional[Sequence]) – extra variables (genes) to be added to the auto-selected ones as the variable(gene)-nodes.

Returns

  • came_inputs (a dict containing CAME inputs)

  • (adata1, adata2) (a tuple of the preprocessed AnnData objects)
