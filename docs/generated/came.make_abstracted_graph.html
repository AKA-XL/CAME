came.make_abstracted_graph¶

came.make_abstracted_graph(obs_labels1, obs_labels2, var_labels1, var_labels2, avg_expr1, avg_expr2, df_var_links, tags_obs=('', ''), tags_var=('', ''), key_weight='weight', cut_ov=0.0, norm_mtd_ov='zs', ov_norm_first=True, global_adjust_ov=True, global_adjust_vv=True, vargroup_filtered='filtered', **kwds)¶

Compute and make the abstracted graph from expression matrices and the linkage weights between homologous genes

Parameters

  • obs_labels1 (Sequence) – group labels of the reference and query observations (cells), respectively.

  • obs_labels2 (Sequence) – group labels of the reference and query observations (cells), respectively.

  • var_labels1 (Series) – group labels of the reference and query variables (genes), respectively.

  • var_labels2 (Series) – group labels of the reference and query variables (genes), respectively.

  • avg_expr1 (DataFrame) – averaged expression matrix of the reference data

  • avg_expr2 (DataFrame) – averaged expression matrix of the query data

  • df_var_links (DataFrame) – the linkage-weights between homologous genes

  • tags_obs – a tuple of two strings, for specifying homologous cel-types from different species. For example, if set tags_obs=('human ', 'mouse '), the result node names (for ‘T cell’) will be ‘human T cell’ and ‘mouse T cell’, respectively.

  • tags_var – a tuple of two strings, for specifying gene modules from different species. For example, if set tags_obs=('human module ', 'mouse module '), the result node names (for gene module ‘1’) will be ‘human module 1’ and ‘mouse module 1’, respectively.

  • key_weight (str) –

    column name in df_var_links, specifying weights between each pair

    of variables.

  • cut_ov (float) – the threshold to cut edges with values lower than it.

  • norm_mtd_ov (Optional[str]) – one of {None, ‘zs’, ‘maxmin’, ‘max’}

  • global_adjust_ov (bool) – whether to globally adjust the weights between the observations and the variables

  • global_adjust_vv (bool) – whether to globally adjust the weights between the observations
