came.utils.preprocess.get_homologues¶

came.utils.preprocess.get_homologues(df_match, vals, cols=None, reverse=False, uniquelist=True, with_null=False)¶

Get the homologous gene of input ones based on the homology-mappings

Parameters

  • df_match (DataFrame) – homologous gene mappings

  • vals (Sequence) – the gene names.

  • cols (Optional[Sequence[str]]) – Two column names as [col_from, col_to], specifying the homology mapping direction.

  • reverse (bool) – whether to reverse the mappings

  • uniquelist (bool) – whether to remove the duplicates from the results.

  • with_null – whether to return those genes without any homologies (copy names)

Returns

  • If uniquelist=True and with_null=False, a single list will be returned;

  • otherwise, returns a tuple of two lists (homo, null)
